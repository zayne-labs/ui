Zayne UI uses a compound component pattern — sub-components that work together under a shared namespace.

## Compound Components

```tsx
import { DropZone } from "@zayne-labs/ui-react/ui/drop-zone";

<DropZone.Root onUpload={handleUpload}>
	<DropZone.Area>
		<p>Drag and drop files here</p>
		<DropZone.Trigger>Select Files</DropZone.Trigger>
	</DropZone.Area>

	<DropZone.FileList>
		{({ fileState }) => (
			<DropZone.FileItem fileState={fileState}>
				<DropZone.FileItemPreview />
				<DropZone.FileItemMetadata />
				<DropZone.FileItemDelete>Remove</DropZone.FileItemDelete>
			</DropZone.FileItem>
		)}
	</DropZone.FileList>
</DropZone.Root>;
```

Each sub-component can be used independently for full control over composition.

## Data Attributes

Every component renders three data attributes:

| Attribute    | Description         | Example                            |
| :----------- | :------------------ | :--------------------------------- |
| `data-scope` | Component family    | `"dropzone"`, `"card"`             |
| `data-part`  | Specific part       | `"root"`, `"area"`, `"title"`      |
| `data-slot`  | Combined identifier | `"dropzone-area"`, `"card-header"` |

Use them for CSS targeting without relying on class names:

```css
[data-scope="dropzone"][data-part="area"] {
	border: 2px dashed;
}

[data-scope="dropzone"][data-part="area"][data-drag-over] {
	background: oklch(0.97 0.01 265);
}
```

## Import Pattern

Components support both namespaced and individual imports:

```tsx
// Namespaced (recommended)
// Individual
import { DropZone, DropZoneRoot } from "@zayne-labs/ui-react/ui/drop-zone";

<DropZone.Root>...</DropZone.Root>;

<DropZoneRoot>...</DropZoneRoot>;
```

## UI vs Utility Components

**UI Components** (`ui/*`) are interactive components with built-in state management and richer composition:

```tsx
import { Card } from "@zayne-labs/ui-react/ui/card";
import { DropZone } from "@zayne-labs/ui-react/ui/drop-zone";
```

Available: `Card`, `Carousel`, `DragScroll`, `DropZone`, `Form`

---

**Utility Components** (`common/*`) are lightweight rendering primitives with no styling:

```tsx
import { For } from "@zayne-labs/ui-react/common/for";
import { Show } from "@zayne-labs/ui-react/common/show";
```

Available: `For`, `Show`, `Switch`, `Await`, `Presence`, `ErrorBoundary`, `Teleport`, `ClientGate`
