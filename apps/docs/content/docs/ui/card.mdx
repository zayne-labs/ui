The Card component provides a structured way to display content in a self-contained container. It's built with composition in mind, allowing you to combine different parts like headers, titles, descriptions, content areas, and footers.

## Installation

```package-install
@zayne-labs/ui-react
```

## Basic Usage

```tsx
import { Card } from "@zayne-labs/ui-react/ui/card";

<Card.Root>
	<Card.Header>
		<Card.Title>Account Overview</Card.Title>
		<Card.Description>Manage your profile and settings</Card.Description>
	</Card.Header>
	<Card.Content>
		<p className="text-sm text-zu-muted-foreground">
			View and update your personal information, security settings, and preferences.
		</p>
	</Card.Content>
	<Card.Footer>
		<Card.Action className="rounded-md bg-zu-primary px-4 py-2 text-white">Edit Profile</Card.Action>
	</Card.Footer>
</Card.Root>;
```

## Component API

All Card components are polymorphic and support the `asChild` pattern for maximum flexibility.

- `Card.Root` - The primary container for the card (renders `<article>` by default)
- `Card.Header` - Container for titles and descriptions (renders `<header>` by default)
- `Card.Title` - The title element within the card header (renders `<h3>` by default)
- `Card.Description` - The description element within the card header (renders `<p>` by default)
- `Card.Content` - The main content area of the card (renders `<div>` by default)
- `Card.Action` - An interactive element within the card (renders `<button>` by default)
- `Card.Footer` - The footer section of the card (renders `<footer>` by default)

## Data Attributes

| Component   | Attribute           | Description                                                |
| :---------- | :------------------ | :--------------------------------------------------------- |
| `Card.Root` | `data-scope="card"` | The component family.                                      |
| All parts   | `data-part`         | Root, Header, Title, Description, Content, Action, Footer. |
| All parts   | `data-slot`         | Combined identifier (e.g., `card-header`).                 |

```css title="styles.css"
[data-slot="card-root"] {
  border-radius: 12px;
  background-color: var(--zu-background);
}

[data-slot="card-title"] {
  font-size: 1.25rem;
  font-weight: 700;
}
```
