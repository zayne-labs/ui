---
title: DragScroll
description: A headless hook for adding mouse drag-to-scroll functionality to containers with optional navigation buttons.
---

The `useDragScroll` hook provides mouse drag-to-scroll behavior for horizontal or vertical scrollable containers. It's completely headless and unstyled, giving you full control over the UI while handling all the complex drag, scroll, and navigation logic.

## Installation

```package-install
@zayne-labs/ui-react
```

## Basic Usage

```tsx
import { useDragScroll } from "@zayne-labs/ui-react/ui/drag-scroll";

function HorizontalScroller() {
	const { propGetters } = useDragScroll();

	return (
		<div className="relative">
			<button {...propGetters.getBackButtonProps()}>←</button>

			<div {...propGetters.getRootProps()} className="flex gap-4 overflow-x-auto p-4">
				{[1, 2, 3, 4, 5].map((item) => (
					<div
						key={item}
						{...propGetters.getItemProps()}
						className="flex h-40 w-64 shrink-0 items-center justify-center rounded-xl border border-zu-primary/20 bg-zu-primary/10"
					>
						Item {item}
					</div>
				))}
			</div>

			<button {...propGetters.getNextButtonProps()}>→</button>
		</div>
	);
}
```

## Return Values

<Accordions>
	<Accordion title="propGetters">
		Functions that return a collection of props to be spread onto your elements. - `getRootProps`: Props
		for the scrollable container. - `getItemProps`: Props for each scrollable item. -
		`getBackButtonProps`: Props for the previous navigation button. - `getNextButtonProps`: Props for the
		next navigation button.
	</Accordion>
	<Accordion title="containerRef">
		A standard React ref object containing the scrollable container element.
	</Accordion>
	<Accordion title="storeApi">The raw Zustand-like store API for advanced state management.</Accordion>
	<Accordion title="useDragScrollStore">
		A hook-based selector for subscribing to specific pieces of the drag-scroll state.
	</Accordion>
</Accordions>

## Data Attributes

The hook automatically applies data attributes to the root element for styling based on its state:

| Attribute       | Value               | Description                                       |
| :-------------- | :------------------ | :------------------------------------------------ |
| `data-scope`    | `"drag-scroll"`     | Identifies the component family.                  |
| `data-part`     | `"root"`            | Identifies the root container.                    |
| `data-dragging` | `"true" \| "false"` | Whether the container is currently being dragged. |

```css title="styles.css"
[data-scope="drag-scroll"][data-part="root"][data-dragging="true"] {
  cursor: grabbing;
  user-select: none;
}
```
