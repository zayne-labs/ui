The Carousel component provides a fully-featured image and content slider with support for automatic sliding, navigation buttons, indicators, captions, and custom rendering. It's built with composition in mind and offers fine-grained control over behavior and appearance.

## Installation

```package-install
@zayne-labs/ui-react
```

## Basic Usage

```tsx
import { Carousel } from "@zayne-labs/ui-react/ui/carousel";

const items = [
	{ src: "https://example.com/image1.jpg", title: "Slide 1" },
	{ src: "https://example.com/image2.jpg", title: "Slide 2" },
	{ src: "https://example.com/image3.jpg", title: "Slide 3" },
];

<Carousel.Root images={items}>
	<Carousel.ItemList>
		{({ image }) => (
			<Carousel.Item>
				<img src={image.src} alt={image.title} className="h-96 w-full rounded-xl object-cover" />
			</Carousel.Item>
		)}
	</Carousel.ItemList>

	<Carousel.Controls />

	<Carousel.IndicatorList>
		{({ index }) => <Carousel.Indicator currentIndex={index} />}
	</Carousel.IndicatorList>
</Carousel.Root>;
```

### Other Components

The following components accept standard HTML props:

- `Carousel.ItemList` - Container for carousel items (accepts `className`)
- `Carousel.Item` - Individual carousel item (accepts `className`)
- `Carousel.IndicatorList` - Container for indicators (accepts `className`)
- `Carousel.Caption` - Caption overlay (accepts `className` and polymorphic `as` prop)

## Navigation Patterns

The Carousel supports flexible icon configuration to handle both symmetrical and asymmetrical designs:

<Accordions>
  <Accordion title="Separate Icons">
    ```tsx
    <Carousel.Controls
      icon={{
        prev: <PrevIcon />,
        next: <NextIcon />,
      }}
    />
    ```
  </Accordion>
  <Accordion title="Single Icon Pattern">
    Provide one icon and specify its type. The library will handle rotating the icon for the opposite direction.

    ```tsx
    <Carousel.Controls
      icon={{
        icon: <ArrowIcon />,
        iconType: "nextIcon", // Prev button will be rotated 180deg
      }}
    />
    ```

  </Accordion>
</Accordions>

## Data Attributes

| Component            | Attribute               | Description                              |
| :------------------- | :---------------------- | :--------------------------------------- |
| `Carousel.Root`      | `data-scope="carousel"` | The component family.                    |
| `Carousel.Item`      | `data-part="item"`      | Individual slide element.                |
| `Carousel.Indicator` | `data-active`           | Present on the active slide's indicator. |

```css title="styles.css"
[data-scope="carousel"] [data-active] {
  background-color: var(--zu-primary);
  width: 2rem;
}
```
