Zayne UI ships with minimal structural styles for layout only. Override with `className` or wipe entirely with the `unstyled` prop.

## Design Philosophy

Components ship with minimal structural styles — override with `className` or use the `unstyled` prop for full control. Data attributes are available for CSS targeting.

## Using with Tailwind CSS

Zayne UI is built for Tailwind CSS v4. Import the preset in your global CSS as shown in the [Installation Guide](/docs/installation#tailwind-css-integration-v4).

### Theme Tokens

The preset defines these CSS custom properties:

```css title="@zayne-labs/ui-react/css/theme.css"
@theme {
  --color-zu-foreground: oklch(0.145 0 0);
  --color-zu-accent: oklch(0.967 0.001 286.375);
  --color-zu-accent-foreground: oklch(0.205 0 0);
  --color-zu-muted-foreground: oklch(0.556 0 0);
  --color-zu-destructive: oklch(0.577 0.245 27.325);
  --color-zu-primary: oklch(0.21 0.04 265.75);
  --color-zu-primary-foreground: oklch(0.985 0 0);
  --color-zu-ring: oklch(0.705 0.015 286.067);
}

:where(.dark, [data-theme="dark"]) {
  --color-zu-foreground: oklch(0.985 0 0);
  --color-zu-primary: oklch(0.92 0.004 286.32);
  /* ... */
}
```

<Callout>
	Dark mode activates automatically with the `.dark` class or `data-theme="dark"` attribute.
</Callout>

### Animations

The preset also includes named animations you can use directly:

| Token                  | Description                               |
| :--------------------- | :---------------------------------------- |
| `animate-shake`        | Error shake (used by `Form.ErrorMessage`) |
| `animate-fade-up`      | Fade in from below                        |
| `animate-fade-down`    | Fade in from above                        |
| `animate-files-in`     | File item entry (used by `DropZone`)      |
| `animate-progress-out` | Progress bar exit (used by `DropZone`)    |

### Utilities

The preset adds a `scrollbar-hidden` utility that hides scrollbars cross-browser:

```tsx
<DragScroll.List className="scrollbar-hidden flex gap-4 overflow-x-auto" />
```

### Example

```tsx
import { DropZone } from "@zayne-labs/ui-react/ui/drop-zone";

<DropZone.Root className="w-full max-w-xl">
	<DropZone.Area
		className="flex flex-col items-center rounded-lg border-2 border-dashed
    border-gray-300 p-12 transition-colors hover:bg-gray-50
    data-drag-over:border-zu-primary data-drag-over:bg-zu-primary/5"
	>
		<p className="font-medium text-gray-700">Drop files here</p>
		<DropZone.Trigger className="mt-4 rounded-md bg-zu-primary px-6 py-2 text-white hover:opacity-90">
			Select Files
		</DropZone.Trigger>
	</DropZone.Area>
</DropZone.Root>;
```

## Using Without Tailwind

### Pre-compiled Styles

Import the compiled CSS in your root file:

```tsx
import "@zayne-labs/ui-react/style.css";
```

### Vanilla CSS via Data Attributes

Every component renders `data-scope`, `data-part`, and `data-slot` attributes you can target:

```css title="styles.css"
[data-scope="drag-scroll"][data-part="root"] {
  display: flex;
  overflow-x: auto;
  gap: 1rem;
  cursor: grab;
}

[data-scope="drag-scroll"][data-part="root"]:active {
  cursor: grabbing;
}

/* Or use the combined slot */
[data-slot="drag-scroll-root"] {
  scrollbar-width: none;
}
```
